craftr.dsl.rewrite

Rewrite Craftr DSL code to pure Python code.

Grammar Objects

class Grammar()

Grammar settings.

SyntaxError Objects

@dataclass
class SyntaxError(Exception)

Specialized Craftr DSL syntax error (the internal SyntaxError class is weird).

If the termcolor module is installed, the error message will be color coded.

RewriteResult Objects

@dataclass
class RewriteResult()

The result of rewriting Craftr DSL code to pure Python code.

Rewriter Objects

class Rewriter()

This class rewrites Craftr DSL code to pure Python code. Closures are extracted from the code and replaced with whitespace where appropriate to keep line and column numbers of the code in tact as much as possible (not always fully accurate).

__init__

 | __init__(text: str, filename: str, grammar: t.Optional[Grammar] = None) -> None

Arguments

  • text: The Craftr DSL code to parse and turn into an AST-like structure.
  • filename: The filename where the DSL code is from.

rewrite

 | rewrite() -> RewriteResult

Rewrite the code and return the RewriteResult. This can be interpreted by the craftr.dsl.transpiler.ClosureRewriter to re-inject the code for closures.
